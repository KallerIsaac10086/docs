## 概述

LawChat 提供了强大的外部工具接入能力（如法律案例搜索、计算器等）。管理员可以通过后台灵活地启用或禁用这些工具，并配置其运行参数。

## 工具列表

后台工具管理页面显示了所有可用的外部插件及其当前状态：

- **工具名称**：插件的系统内部标识。
- **显示名称**：用户在界面上看到的名称。
- **描述信息**：说明该工具的具体功能及应用场景。
- **当前状态**：显示该工具是否已在系统中启用。

## 管理操作

管理员可以在后台界面对插件执行以下管理任务：

### 1. 启停控制
- **一键启用/禁用**：管理员可以随时开关特定工具的访问权限，确保服务的稳定性或响应业务调整。

### 2. 参数定义
- **参数详情查看**：查看工具运行所需的具体参数定义（基于 JSON Schema 格式），如输入字段类型、是否必填等。

## 关键字段

| 字段 | 说明 |
|------|------|
| ID | 插件工具的唯一 ID |
| 系统标识 | 用于代码逻辑中调用的内部名称 |
| 显示名称 | 在前端用户交互界面展示的名称 |
| 功能描述 | 该工具的主要作用说明 |
| 参数配置 | 定义工具如何接收用户输入 |
| 启用状态 | 决定用户是否可以使用该工具 |
