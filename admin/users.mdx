## 概述

管理员可以在后台查看、搜索并管理所有注册用户的信息。用户管理模块是系统运营的核心工具。

## 用户列表

后台提供完整的用户列表视图，支持以下关键信息的快速查看：

- **用户基本信息**：用户名、邮箱、手机号等。
- **账户状态**：正常、已禁用、已挂起或待激活。
- **会员等级**：Free、Pro 或 Max。
- **账户余额/积分**：实时查看用户当前的积分余额。
- **到期时间**：会员服务的有效期。

## 用户操作

通过后台界面，管理员可以执行以下操作：

### 1. 状态管理
- **禁用用户**：禁止违规用户登录。
- **激活用户**：手动激活新注册或已过期的账户。

### 2. 权限管理
- **调整会员等级**：根据业务需要手动升级或降级用户的会员层级。
- **设置用量限制**：针对特定用户调整其每日或总计的 Token 消耗上限。

### 3. 数据审计
- **查看登录日志**：审计用户的访问历史。
- **查看交易记录**：核对用户的充值与积分消耗明细。

## 字段说明

| 字段 | 说明 |
|------|------|
| 用户 ID | 系统内唯一标识符 |
| 账户状态 | 控制用户是否可正常使用服务 |
| 会员等级 | 决定用户可访问的模型及费率 |
| 积分余额 | 用户当前可用于对话的积分总额 |
| 用量限制 | 包括每日输入/输出 Token 限制、对话次数限制等 |
