## 概述

数据分析模块为管理员提供了全方位的系统运行指标监控，帮助深入了解用户行为、业务增长及资源消耗情况，为运营决策提供数据支持。

## 用户统计分析

管理员可以从多维度分析用户增长与活跃情况：

- **增长趋势**：监控每日新增用户、总用户量及活跃用户的变化情况。
- **用户分布**：分析不同会员等级（Free、Pro、Max）的用户比例，以及账户状态（正常、禁用等）的分布。

## 用量统计监控

系统实时记录并分析各项资源的消耗数据：

### 1. Token 消耗统计
- **总量监控**：查看全站每日消耗的总 Token 数量。
- **输入输出比**：分析用户输入与 AI 生成内容的比例，优化上下文长度策略。
- **日均用量**：计算平均每日消耗量，预估后端 API 成本。

### 2. 对话活跃度
- **对话次数统计**：监控系统每日发起的总对话次数。
- **热门时段分析**：识别用户使用高峰期，合理配置服务器资源。

## 财务与收入分析

针对盈利项目的核心数据分析：

- **收入统计**：查看不同时间段内的订阅收入总额。
- **套餐热度**：分析各级别套餐的购买转化率，优化定价策略。
- **交易流水**：审计所有充值与消费行为，确保财务安全。

## 数据展示

后台采用可视化图表展示数据，包括但不限于：
- 趋势折线图（用于用户增长与 Token 消耗）。
- 比例饼图（用于会员分布与套餐热度）。
- 详细数据表格（用于具体交易与操作审计）。
