## 概述

数据分析模块提供系统使用情况的统计和分析功能。

## 用户统计

### 用户增长趋势

```bash
curl -X GET http://localhost:7878/manage/analytics/users/growth \
  -H "Authorization: Bearer ADMIN_TOKEN"
```

### 响应示例

```json
{
  "data": [
    {
      "date": "2024-01-01",
      "total_users": 100,
      "new_users": 10,
      "active_users": 80
    },
    {
      "date": "2024-01-02",
      "total_users": 110,
      "new_users": 10,
      "active_users": 85
    }
  ]
}
```

### 用户分布

```bash
curl -X GET http://localhost:7878/manage/analytics/users/distribution \
  -H "Authorization: Bearer ADMIN_TOKEN"
```

### 响应示例

```json
{
  "by_tier": {
    "free": 50,
    "pro": 40,
    "max": 10
  },
  "by_status": {
    "normal": 95,
    "disabled": 5
  }
}
```

## 时间范围

### 查询参数

| 参数 | 说明 | 示例 |
|------|------|------|
| start_date | 开始日期 | 2024-01-01 |
| end_date | 结束日期 | 2024-01-31 |
| period | 时间周期 | daily/weekly/monthly |

### 示例查询

```bash
curl -X GET "http://localhost:7878/manage/analytics/users/growth?start_date=2024-01-01&end_date=2024-01-31&period=daily" \
  -H "Authorization: Bearer ADMIN_TOKEN"
```

## 导出数据

### 导出为 CSV

```bash
curl -X GET "http://localhost:7878/manage/analytics/export?format=csv" \
  -H "Authorization: Bearer ADMIN_TOKEN" \
  -o analytics.csv
```

### 导出为 JSON

```bash
curl -X GET "http://localhost:7878/manage/analytics/export?format=json" \
  -H "Authorization: Bearer ADMIN_TOKEN" \
  -o analytics.json
```
