## 环境准备

在开始之前，请确保你已经安装了以下工具：

### 必需软件
- **Python 3.12+** - 后端运行环境
- **Node.js 18+** - 前端运行环境
- **npm 或 pnpm** - 包管理器

### 可选软件
- **PostgreSQL** - 生产环境数据库（默认使用 SQLite）
- **Xcode 15+** - iOS 开发（macOS）
- **Android Studio** - Android 开发

## 第一步：克隆项目

```bash
git clone <repository-url>
cd llama-python-webui
```

## 第二步：启动后端

### 1. 进入后端目录
```bash
cd backend
```

### 2. 创建虚拟环境（推荐）
```bash
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
.venv\Scripts\activate     # Windows
```

### 3. 安装依赖
```bash
pip install -r requirements.txt
```

### 4. 初始化数据库
```bash
python init_db.py
```

### 5. 配置模型
编辑 `backend/api/models/models.json`，添加你的模型配置：

```json
{
  "providers": {
    "YourProvider": {
      "type": "openai",
      "base_url": "http://your-model-server/v1",
      "api_key": "your-api-key",
      "models": ["your-model-name"],
      "ability": {
        "think": "dynamic",
        "vision": false
      },
      "model_info": {
        "your-model-name": {
          "id": "unique-uuid",
          "display_name": "你的模型",
          "developer": "你的开发者",
          "tier": "free",
          "max_context": "32k",
          "system_prompt": "系统提示词",
          "default_params": {
            "temperature": 0.7,
            "max_tokens": 4096
          },
          "status": "active"
        }
      }
    }
  }
}
```

### 6. 启动后端服务
```bash
python main.py
```

或使用 uvicorn：
```bash
uvicorn main:app --host 0.0.0.0 --port 7878 --reload
```

后端服务将在 `http://localhost:7878` 启动。

## 第三步：启动前端

### 1. 进入前端目录
```bash
cd frontend
```

### 2. 安装依赖
```bash
npm install
```

### 3. 启动开发服务器
```bash
npm run dev
```

前端服务将在 `http://localhost:6006` 启动。

## 第四步：验证安装

### 1. 访问前端
打开浏览器访问 `http://localhost:6006`

### 2. 访问后端 API 文档
打开浏览器访问 `http://localhost:7878/docs` 查看 Swagger UI

### 3. 注册账户
在前端页面注册一个测试账户

### 4. 开始聊天
选择一个模型，开始与 AI 对话

## 默认账户
