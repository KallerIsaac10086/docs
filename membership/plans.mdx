## 模型计费对比

我们针对法律业务的不同复杂度，设计了三种模型层级。请根据您的案件类型选择最合适的服务。

| 模型类别      | Free                     | Pro                            | Max                            |
| :-------- | :----------------------- | :----------------------------- | :----------------------------- |
| **适用场景**  | 基础法规查询、日常咨询              | 复杂案情梳理、法律文书起草、多轮案情推演           | 疑难案件逻辑推理、诉讼策略分析、深度判例研究         |
| **基础扣费**  | **2 分 / 次**              | **0.1 分 / 次**                  | **0.2 分 / 次**                  |
| **计费模式**  | 固定扣费 (触发熔断后按量)           | **基础费 + Token 阶梯系数**           | **基础费 + Token 阶梯系数**           |
| **长文本支持** | 不限长度 (适合上传长篇合同)          | 不限长度 (系数随案情描述长度动态调整)           | 不限长度 (系数随证据链复杂度动态调整)           |
| **计费公式**  | `2` (熔断前)                | `(0.1 + Tokens/1000 × k) × 倍率` | `(0.2 + Tokens/1000 × k) × 倍率` |
| **熔断机制**  | 月累计 \> 200k Tokens 后额外计费 | 无                              | 无                              |

### 阶梯系数 (k) 详情表

法律案件通常涉及大量事实描述和证据材料。当单次请求总 Token 数（含历史对话与上传文档）落入不同区间时，系数 `k` 动态调整：

| Token 范围     | Free 系数 | Pro 系数  | Max 系数  | 法律场景示例             |
| :----------- | :-----: | :-----: | :-----: | :----------------- |
| **0 - 1k**   | -       | **0.8** | **1.2** | 简单法条定义查询 / 简短咨询    |
| **1k - 2k**  | -       | **1.0** | **1.4** | 标准合同条款审查 / 一般纠纷咨询  |
| **2k - 3k**  | -       | **1.2** | **1.6** | 中等复杂度案情陈述 / 律师函起草  |
| **3k - 4k**  | -       | **1.4** | **1.8** | 多份证据材料分析 / 复杂协议修订  |
| **4k - 10k** | -       | **1.6** | **2.5** | 完整案卷梳理 / 诉讼风险评估报告  |
| **\> 10k**   | -       | **2.5** | **5.0** | 超大型并购案分析 / 跨年度案件复盘 |

> **注意**：Free 模型在单次累计消耗超过 **200,000 Tokens** 时，超出部分按 **1 分 / 1000 Tokens** 额外扣除。这通常发生在您上传了大量长篇合同或判决书进行批量阅读时。

## 特殊模式与规则

### 深度思考模式 (Thinking Mode)

在处理**疑难案件**或**复杂逻辑推理**时，您可以启用 **深度思考模式**。AI 将模拟资深律师的思维链（Chain of Thought），逐步拆解法律关系。

由于该模式消耗大量计算资源进行深度推演，该次请求产生的所有计费均 **双倍扣除**。

| 模式状态       | 计费倍率       | 建议用途            |
| :--------- | :--------: | :-------------- |
| 普通模式       | `1.0x`     | 快速查询、常规文书生成     |
| **深度思考模式** | **`2.0x`** | 诉讼策略制定、复杂争议焦点分析 |

### 积分重置规则

系统将于每月 **1 号 0:00** 自动执行积分重置，确保您每月拥有充足的额度处理新案件。

| 规则项        | 说明                   |
| :--------- | :------------------- |
| **重置时间**   | 每月 1 号 0:00          |
| **订阅用户额度** | 重置为 **3500 分**       |
| **剩余积分处理** | 上月剩余积分清零，**不累积**至新月份 |
