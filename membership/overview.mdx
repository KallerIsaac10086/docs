## 概述

LawChat 采用三级会员体系，为不同需求的用户提供灵活的服务方案。

## 会员等级

| 等级 | 说明 | 适用场景 |
|------|------|----------|
| Free | 免费用户 | 个人试用、轻度使用 |
| Pro | 专业用户 | 专业人士、高频使用 |
| Max | 高级用户 | 企业用户、团队协作 |

## 会员权益

### Free 用户

- 基础模型访问
- 每日对话次数限制
- 每日 Token 限制
- 基础功能支持

### Pro 用户

- 所有 Free 权益
- Pro 模型访问
- 更高的每日对话次数
- 更高的每日 Token 限制
- 优先客服支持
- 云端同步

### Max 用户

- 所有 Pro 权益
- Max 模型访问
- 无限对话次数
- 无限 Token 使用
- 专属客服支持
- 优先功能体验
- 团队协作功能

## 会员状态

| 状态 | 说明 |
|------|------|
| active | 会员有效 |
| expired | 会员已过期 |
| pending | 会员待激活 |
| cancelled | 会员已取消 |

## 用量限制

### Token 限制

- **输入 Token**: 用户输入的 Token 数量
- **输出 Token**: AI 生成的 Token 数量
- **每日限制**: 每日可使用的 Token 总量
- **总限制**: 总计可使用的 Token 数量

### 对话次数限制

- **每日对话限制**: 每日可发起的对话次数
- **总对话限制**: 总计可发起的对话次数

### 请求次数限制

- **每日请求限制**: 每日可发起的 API 请求次数
- **总请求限制**: 总计可发起的 API 请求次数

## 会员过期

会员过期后：

1. 自动降级为 Free 用户
2. 保留历史对话记录
3. 限制访问 Pro/Max 模型
4. 恢复默认用量限制

## 会员升级

用户可以随时升级会员：

1. 在会员页面选择更高等级的套餐
2. 完成支付
3. 立即生效
4. 剩余时间累加到新套餐

## 会员续费

订阅套餐支持自动续费：

- **月付**: 每月自动扣费
- **年付**: 每年自动扣费
- **一次性**: 一次性购买，不自动续费

## 资源包

除了会员套餐，用户还可以购买资源包：

### 资源类型

| 类型 | 说明 |
|------|------|
| tokens | Token 资源包 |
| chats | 对话次数包 |
| requests | 请求次数包 |

### 资源包特点

- 按需购买，灵活补充
- 可叠加使用
- 支持过期时间
- 支持重置周期

## 余额支付

用户可以使用余额支付：

1. 充值到账户余额
2. 使用余额购买套餐或资源包
3. 余额不足时提示充值

## 代金券

系统支持代金券：

- 管理员创建代金券
- 用户兑换代金券
- 支持使用次数限制
- 支持过期时间

## 查询会员状态

### API 查询

```bash
curl -X GET http://localhost:7878/api/membership/status \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### 响应示例

```json
{
  "tier": "pro",
  "expires_at": "2024-12-31T23:59:59Z",
  "limits": {
    "daily_chat_limit": 100,
    "daily_token_limit": 100000,
    "total_chat_limit": 1000,
    "total_token_limit": 1000000
  },
  "usage": {
    "daily_chats_used": 45,
    "daily_tokens_used": 45000,
    "total_chats_used": 234,
    "total_tokens_used": 234000
  }
}
```
