## 通用问题

### Q: 如何重置管理员密码？

A: 
1. 登录管理后台
2. 进入用户设置
3. 修改管理员密码

### Q: 如何备份数据库？

A: 
1. 停止后端服务
2. 复制 `database.db` 文件
3. 保存到安全位置

### Q: 如何升级会员？

A: 
1. 登录用户账户
2. 进入会员页面
3. 选择更高等级的套餐
4. 完成支付

### Q: 如何兑换代金券？

A: 
1. 登录用户账户
2. 进入会员页面
3. 找到代金券兑换入口
4. 输入代金券代码
5. 点击兑换

### Q: 如何查看用量统计？

A: 
1. 登录用户账户
2. 进入设置页面
3. 查看用量统计信息

## 后端问题

### Q: 后端无法启动？

A: 检查以下几点：
1. Python 版本是否为 3.12+
2. 依赖是否正确安装
3. 端口 7878 是否被占用
4. 数据库连接是否正常

### Q: 模型调用失败？

A: 检查以下几点：
1. `models.json` 配置是否正确
2. API Key 是否有效
3. 模型提供商服务是否可访问
4. 查看后端日志获取详细错误信息

### Q: 数据库连接失败？

A: 检查以下几点：
1. `DB_TYPE` 环境变量是否正确
2. PostgreSQL 服务是否运行
3. 连接参数是否正确
4. 数据库用户权限是否足够

### Q: 定时任务不执行？

A: 检查以下几点：
1. 后端服务是否正常运行
2. 查看启动日志是否有错误
3. 检查系统时间是否正确

## 前端问题

### Q: 前端无法启动？

A: 检查以下几点：
1. Node.js 版本是否为 18+
2. 依赖是否正确安装
3. 端口 6006 是否被占用
4. 后端服务是否正常运行

### Q: API 请求失败？

A: 检查以下几点：
1. 后端服务是否运行
2. CORS 配置是否正确
3. 网络连接是否正常
4. 认证 Token 是否有效

### Q: 前端构建失败？

A: 尝试以下步骤：
1. 删除 `node_modules` 和 `.svelte-kit` 目录
2. 重新运行 `npm install`
3. 确保 Node.js 版本符合要求

### Q: 工具无法加载？

A: 检查以下几点：
1. 工具是否在数据库中启用
2. 后端日志中是否有错误
3. 工具注册是否成功

## 会员问题

### Q: 会员过期后无法使用？

A: 
1. 会员过期后自动降级为 Free 用户
2. 可以重新购买套餐升级
3. 历有对话记录会保留

### Q: 订阅扣费失败？

A: 
1. 检查账户余额是否充足
2. 查看交易记录了解失败原因
3. 联系客服寻求帮助

### Q: 代金券无法兑换？

A: 检查以下几点：
1. 代金券代码是否正确
2. 代金券是否已过期
3. 代金券使用次数是否已用完
4. 是否已经使用过该代金券

## 部署问题

### Q: Docker 部署失败？

A: 检查以下几点：
1. Docker 是否正确安装
2. Dockerfile 配置是否正确
3. 网络连接是否正常
4. 查看容器日志

### Q: Nginx 配置错误？

A: 检查以下几点：
1. Nginx 配置文件语法是否正确
2. 端口是否被占用
3. 权限是否正确
4. 使用 `nginx -t` 测试配置

### Q: SSL 证书问题？

A: 
1. 使用 Let's Encrypt 获取免费证书
2. 确保证书路径正确
3. 检查证书是否过期
4. 重启 Nginx 使配置生效

## 移动端问题

### Q: iOS 应用无法构建？

A: 检查以下几点：
1. Xcode 版本是否符合要求
2. 签名配置是否正确
3. Bundle Identifier 是否唯一
4. Provisioning Profile 是否匹配

### Q: Android 应用无法构建？

A: 检查以下几点：
1. Android Studio 版本是否符合要求
2. Keystore 是否存在
3. 签名密码是否正确
4. Key Alias 是否匹配

## 性能问题

### Q: 响应速度慢？

A: 优化建议：
1. 使用更快的模型
2. 启用缓存
3. 优化数据库查询
4. 使用 CDN 加速静态资源

### Q: 内存占用过高？

A: 优化建议：
1. 限制并发请求数
2. 优化模型加载
3. 定期清理缓存
4. 使用内存监控工具
