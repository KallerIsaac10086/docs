## 环境要求

- Python 3.12+
- SQLite (默认) 或 PostgreSQL
- 2GB+ 内存
- 10GB+ 磁盘空间

## 安装步骤

### 1. 克隆项目

```bash
git clone <repository-url>
cd llama-python-webui/backend
```

### 2. 创建虚拟环境

```bash
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
.venv\Scripts\activate     # Windows
```

### 3. 安装依赖

```bash
pip install -r requirements.txt
```

### 4. 配置数据库

#### 使用 SQLite（默认）

无需额外配置，数据库文件将自动创建。

#### 使用 PostgreSQL

设置环境变量：

```bash
export DB_TYPE=postgresql
export DB_HOST=your-host
export DB_PORT=5432
export DB_USER=your-user
export DB_PASS=your-password
export DB_NAME=your-database
```

或创建 `.env` 文件：

```env
DB_TYPE=postgresql
DB_HOST=localhost
DB_PORT=5432
DB_USER=lawchat
DB_PASS=your-password
DB_NAME=lawchat
```

### 5. 初始化数据库

```bash
python init_db.py
```

### 6. 配置模型

编辑 `api/models/models.json`，添加你的模型配置。

### 7. 配置环境变量

创建 `.env` 文件：

```env
SECRET_KEY=your-secret-key-here
```

**重要**: 生产环境必须设置强密钥！

### 8. 启动服务

#### 开发模式

```bash
python main.py
```

或使用 uvicorn：

```bash
uvicorn main:app --host 0.0.0.0 --port 7878 --reload
```

#### 生产模式

```bash
uvicorn main:app --host 0.0.0.0 --port 7878 --workers 4
```

## 使用 systemd 管理

### 创建服务文件

```bash
sudo nano /etc/systemd/system/lawchat-backend.service
```

### 服务配置

```ini
[Unit]
Description=LawChat Backend Service
After=network.target

[Service]
Type=simple
User=your-user
WorkingDirectory=/path/to/llama-python-webui/backend
Environment="PATH=/path/to/llama-python-webui/backend/.venv/bin"
ExecStart=/path/to/llama-python-webui/backend/.venv/bin/python main.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

### 启动服务

```bash
sudo systemctl daemon-reload
sudo systemctl enable lawchat-backend
sudo systemctl start lawchat-backend
```

### 查看日志

```bash
sudo journalctl -u lawchat-backend -f
```

## 使用 Docker 部署

### 创建 Dockerfile

```dockerfile
FROM python:3.12-slim

WORKDIR /app

RUN apt-get update && apt-get install -y \
    gcc \
    && rm -rf /var/lib/apt/lists/*

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

RUN python init_db.py

EXPOSE 7878

CMD ["uvicorn", "main:app", "--host", "0.0.0.0", "--port", "7878"]
```

### 构建镜像

```bash
docker build -t lawchat-backend .
```

### 运行容器

```bash
docker run -d \
  --name lawchat-backend \
  -p 7878:7878 \
  -v $(pwd)/database.db:/app/database.db \
  -e SECRET_KEY=your-secret-key \
  lawchat-backend
```

### 使用 Docker Compose

创建 `docker-compose.yml`：

```yaml
version: '3.8'

services:
  backend:
    build: .
    ports:
      - "7878:7878"
    volumes:
      - ./database.db:/app/database.db
    environment:
      - SECRET_KEY=your-secret-key
      - DB_TYPE=postgresql
      - DB_HOST=postgres
      - DB_PORT=5432
      - DB_USER=lawchat
      - DB_PASS=lawchat-password
      - DB_NAME=lawchat
    depends_on:
      - postgres

  postgres:
    image: postgres:15
    environment:
      - POSTGRES_USER=lawchat
      - POSTGRES_PASSWORD=lawchat-password
      - POSTGRES_DB=lawchat
    volumes:
      - postgres_data:/var/lib/postgresql/data

volumes:
  postgres_data:
```

启动服务：

```bash
docker-compose up -d
```

## Nginx 反向代理

### 配置文件

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://localhost:7878;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        # WebSocket 支持
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}
```

### 重启 Nginx

```bash
sudo nginx -t
sudo systemctl reload nginx
```

## 性能优化

### 1. 使用 Gunicorn

```bash
pip install gunicorn
gunicorn main:app -w 4 -k uvicorn.workers.UvicornWorker --bind 0.0.0.0:7878
```

### 2. 启用缓存

在模型配置中启用缓存。

### 3. 数据库优化

- 使用 PostgreSQL 而非 SQLite
- 配置适当的连接池大小
- 定期清理过期数据

## 安全配置

### 1. 设置强密钥

```bash
python -c "import secrets; print(secrets.token_urlsafe(32))"
```

### 2. 配置 CORS

在 `main.py` 中限制 CORS 来源：

```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=["https://your-domain.com"],
    allow_credentials=True,
    allow_methods=["GET", "POST", "PUT", "DELETE"],
    allow_headers=["*"],
)
```

### 3. 启用 HTTPS

使用 Let's Encrypt 获取免费 SSL 证书：

```bash
sudo certbot --nginx -d your-domain.com
```


```bash
sudo systemctl status postgresql
```
