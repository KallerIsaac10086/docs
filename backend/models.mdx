## 配置文件

模型配置存储在 `backend/api/models/models.json`，支持热更新。

## 配置结构

```json
{
  "providers": {
    "ProviderName": {
      "type": "openai",
      "base_url": "http://your-model-server/v1",
      "api_key": "your-api-key",
      "models": ["model-name-1", "model-name-2"],
      "ability": {
        "think": "dynamic",
        "vision": false
      },
      "model_info": {
        "model-name-1": {
          "id": "unique-uuid",
          "display_name": "显示名称",
          "developer": "开发者",
          "tier": "free",
          "max_context": "32k",
          "system_prompt": "系统提示词",
          "system_prompt_tools_enabled": "工具开启时的提示词",
          "system_prompt_tools_disabled": "工具关闭时的提示词",
          "default_params": {
            "temperature": 0.7,
            "max_tokens": 4096
          },
          "status": "active"
        }
      }
    }
  }
}
```

## 配置字段说明

### Provider 配置

| 字段 | 类型 | 说明 |
|------|------|------|
| type | string | 提供商类型 (openai) |
| base_url | string | API 基础 URL |
| api_key | string | API Key |
| models | array | 模型名称列表 |
| ability | object | 模型能力 |

### Ability 配置

| 字段 | 类型 | 说明 |
|------|------|------|
| think | string | 思考能力 (dynamic) |
| vision | boolean | 视觉能力 |

### Model Info 配置

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 唯一 UUID |
| display_name | string | 显示名称 |
| developer | string | 开发者 |
| tier | string | 会员等级 (free/pro/max) |
| max_context | string | 最大上下文 (如 32k, 128k) |
| system_prompt | string | 默认系统提示词 |
| system_prompt_tools_enabled | string | 工具开启时的提示词 |
| system_prompt_tools_disabled | string | 工具关闭时的提示词 |
| default_params | object | 默认参数 |
| status | string | 状态 (active/inactive) |

### Default Params 配置

| 字段 | 类型 | 说明 |
|------|------|------|
| temperature | number | 温度 (0-2) |
| max_tokens | number | 最大 Token 数 |
| top_p | number | 核采样 |
| presence_penalty | number | 存在惩罚 |
| frequency_penalty | number | 频率惩罚 |

## 会员等级

| 等级 | 说明 |
|------|------|
| free | 免费用户可用 |
| pro | Pro 及以上用户可用 |
| max | 仅 Max 用户可用 |

## 热更新

修改 `models.json` 后，重启后端服务即可生效：

```bash
# 重启后端
python main.py
```

或通过管理后台的模型管理界面进行配置。

## 安全注意事项

1. **API Key 保护**: `models.json` 包含敏感的 API Key，请确保文件权限正确
2. **不要提交到版本控制**: 将 `models.json` 添加到 `.gitignore`
3. **生产环境**: 使用环境变量或密钥管理服务存储 API Key
