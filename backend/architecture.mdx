## 应用入口 (main.py)

### 生命周期管理

后端使用 FastAPI 的 lifespan 上下文管理器来管理应用生命周期：

```python
@asynccontextmanager
async def lifespan(app: FastAPI):
    # 启动时
    init_db()                    # 初始化数据库
    ensure_model_ids_and_status() # 确保模型配置
    initialize_retrieval_tools() # 初始化检索工具
    initialize_all_tools()       # 初始化所有工具
    
    # 启动定时任务
    maintenance_task = asyncio.create_task(run_daily_maintenance_task())
    billing_task = asyncio.create_task(run_hourly_subscription_billing())
    cleanup_task = asyncio.create_task(run_pending_transaction_cleanup())
    
    yield
    
    # 关闭时
    maintenance_task.cancel()
    billing_task.cancel()
    cleanup_task.cancel()
```

## 数据库

### 数据库配置 (database.py)

支持 SQLite 和 PostgreSQL 两种数据库：

```python
# SQLite（默认）
DATABASE_URL = "sqlite:///./database.db"

# PostgreSQL（环境变量配置）
DB_TYPE = os.getenv("DB_TYPE", "sqlite")
if DB_TYPE == "postgresql":
    DB_HOST = os.getenv("DB_HOST", "localhost")
    DB_PORT = os.getenv("DB_PORT", "5432")
    DB_USER = os.getenv("DB_USER", "user")
    DB_PASS = os.getenv("DB_PASS", "password")
    DB_NAME = os.getenv("DB_NAME", "lawchat")
    DATABASE_URL = f"postgresql://{DB_USER}:{DB_PASS}@{DB_HOST}:{DB_PORT}/{DB_NAME}"
```

### 数据库模型 (models_db.py)

核心数据表：

- `users` - 用户表
- `membership_plans` - 会员套餐表
- `user_subscriptions` - 用户订阅表
- `payment_transactions` - 支付交易表
- `vouchers` - 代金券表
- `cloud_chats` - 云端对话表
- `private_chats` - 私有对话表
- `usage_stats` - 用量统计表
- `tools` - 工具表
- `user_resource_packs` - 用户资源包表
- `resource_products` - 资源产品表

### 数据库初始化 (init_db.py)

```bash
python init_db.py
```

初始化步骤：
1. 创建所有数据表
2. 创建默认管理员账户
3. 初始化默认套餐
4. 初始化工具数据

## 定时任务

### 每日维护任务

执行时间：北京时间每天 00:00 (UTC 16:00)

功能：
- 重置所有用户的每日用量统计
- 检查并处理过期的会员
- 检查并处理过期的资源包

### 每小时订阅扣费

执行时间：每小时执行一次

功能：
- 处理订阅用户的自动续费扣款

### 待处理交易清理

执行时间：每60秒执行一次

功能：
- 清理30分钟前创建仍处于 pending 状态的支付交易
